# InAmigos NGO Awareness Webpage

A responsive NGO awareness website created using **HTML, CSS, and JavaScript** to showcase the mission, projects, and social impact of **InAmigos Foundation**.

## 🌟 Features

* Modern and responsive design
* NGO introduction section
* Ongoing projects showcase
* Social impact statistics
* Gallery section
* Call-to-action for volunteers and supporters
* Smooth scrolling and animations
* Mobile-friendly layout

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons
* Google Fonts
